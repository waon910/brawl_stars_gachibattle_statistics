about this project

データに基づく戦略で、勝利をつかむ。

本サイトでは Brawl Stars のガチバトル・パワーリーグの戦績を独自に収集し、統計的なアプローチで勝率・使用率を分析しています。 データの出どころや集計ポリシー、そしてプロランク到達の猛者たちをご紹介します。

mission

サイトについて

ガチバトルの膨大なログを可視化し、プレイヤーが「いつ」「どのマップで」「どの編成を選ぶべきか」を素早く判断できるよう支援することが私たちの目的です。 サイトの構成や集計方針を以下にまとめています。

データソース

Supercell 公式 API から取得した戦績を、独自のバトルログ収集基盤で毎日更新しています。トロ帯ごと・マップごとに精緻な統計を生成し、過去30日間のトレンドを追跡しています。

集計ポリシー

ダイヤモンド帯以上の対戦を対象とし、勝率のばらつきを抑えるため Beta-Binomial モデルによる信頼区間を採用。実戦に活かしやすい指標を重視しています。

更新サイクル

データは1時間おきに同期され、バトルログの品質を維持するため重複排除やプレイヤー情報のクレンジングを実施しています。

insight

ランク帯ごとの試合ログ分布

各ランク帯で収集できたバトルログの分布を可視化し、データ量の偏りを随時監視しています。プロ帯に近づくほどログが希少になりやすいため、特に高ランク帯のデータ確保を強化しています。

近日中にインタラクティブなチャートを公開予定です。現在はデータ収集と可視化の改善を進めています。
